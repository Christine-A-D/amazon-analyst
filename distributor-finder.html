Overview
The problem it solves

Finding legitimate distributors and wholesalers for Amazon sourcing is one of the most time-consuming parts of the job. Manually searching Google, visiting websites, and recording contact details can take an entire day — for just a handful of leads.

This tool automates the entire process. By combining the Google Maps API with Oxylabs web scraping, it pulls distributor listings by keyword and location at scale — generating hundreds of qualified leads in a single run.

How it works
Step-by-step process
01
Keyword & location input
The script takes a product category keyword (e.g. "pet supplies distributor") and a target location. These are passed into the Google Maps API to search for matching businesses at scale.
02
Google Maps API scrape
Using the Places API, the script pulls business listings — name, address, phone number, website, and ratings — across multiple pages of results automatically.
03
Website scraping via Oxylabs
For each business found, Oxylabs proxies are used to visit the website and extract additional contact details — emails, product categories, and wholesale inquiry pages.
04
Data cleaning with Pandas
Raw results are processed with Pandas — duplicates removed, fields normalized, and leads scored by relevance. The final output is a clean, ready-to-use spreadsheet.
05
Export to CSV / Google Sheets
The cleaned lead list is exported as a CSV and optionally pushed directly into a Google Sheet for easy review and outreach tracking.
